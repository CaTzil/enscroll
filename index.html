enscroll jQuery Plugin

Customize Scrollbars Easily!

Do you have overflowing boxes with scrollbars on your site? Do you wish you could style those scrollbars to offer your users a consistent look & feel across multiple browsers and platforms? Enscroll is a jQuery plugin that gives you the power to replace those scrollbars rendered by the web browser with ones that you design. Either replace the scrollbars with your own image or use custom CSS to style them anyway you desire.

Make Websites Look the Way You Designed Them

A common problem among web developers when writing markup and stylesheets for site designs is the need to style scrollbars as they appear on the designs. Unfortunately, there is no standard way to accomplish this using only HTML & CSS. So, for now we have to resort to a Javascript solution. This means coding the behavior of scrollbars and the content panes they are scrolling. Who has time to do with this and keep project deadlines?

With the enscroll jQuery plugin, being able to write your own styles for those scrollbars is only a matter of a few lines of Javascript. Here is the short version so you can get started quickly:

The HTML of the Content Pane:

textpop-up
  1. <div class=”scrollbox”>
  2.     <h1>New Scrolling Window</h1>
  3.     <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit...</p>
  4.     <p> ... </p>
  5.     ...
  6. </div>
<div class=”scrollbox”>
	<h1>New Scrolling Window</h1>
	<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit...</p>
	<p> ... </p>
	...
</div>

CSS for the Content Pane and Scrollbars:

textpop-up
  1. .scrollbox {
  2.     overflow: auto; /* This is changed by enscroll, but must be defined */
  3.     width: 368px;   /* The arbitrary width and height of the content pane */
  4.     height: 550px;
  5. }
  6.  
  7. .vertical-track {
  8.     width: 16px;            /* How wide is the scrollbar going to be? */
  9.     background-color: #aaa; /* What color should the vertical track be? */
  10. }
  11.  
  12. .vertical-handle {
  13.     width: 16px;            /* How wide is the scrollbar handle? */
  14.     background-color: #aaf; /* What color should the handle be? */
  15. }
.scrollbox {
	overflow: auto; /* This is changed by enscroll, but must be defined */
	width: 368px;   /* The arbitrary width and height of the content pane */
	height: 550px;
}

.vertical-track {
	width: 16px;            /* How wide is the scrollbar going to be? */
	background-color: #aaa; /* What color should the vertical track be? */
}

.vertical-handle {
	width: 16px;            /* How wide is the scrollbar handle? */
	background-color: #aaf; /* What color should the handle be? */
}

And, Finally the Javascript:

textpop-up
  1. $(document).ready(function() {
  2.     $(”.scrollbox”).enscroll();
  3. });
$(document).ready(function() {
	$(”.scrollbox”).enscroll();
});

Don’t forget that you will also need to include the jQuery script on your page, as well as the enscroll script file.

As you have probably already determined, much of the names and styles used above are arbitrary. The class name of the content pane can be changed, as well as the width and height of the pane. The background colors of the scrollbar track and handle components can also be changed. What is required in the example above is that you provide the content pane with the overflow: auto style. Without this style applied, the plugin will not affect this element. Furthermore, you will need to specify a width for the track and handle.

There are a countless number of designs and styles you can apply to these scrollbars, Start looking at the provided examples to see just a few of the advanced designs that are possible with the enscroll jQuery plugin.

Here's a Few Things You Can Do With enscroll

  • The enscroll element behaves just as any other element with the overflow: auto style applied
  • You have the option of adding only vertical scrolling (the default), only horizontal scrolling, or both
  • Automatically update the position, width and height of the scrollbar elements so that everything adjusts as it should as you add and remove content from the scrolling pane
  • Optionally, keep the scrollbars hidden until the user hovers over the content pane
  • Control how far the scrolling elements scrolls when using the mouse wheel or the arrow keys on the keyboard
  • The minified enscroll Javascript file weighs in at just around 12KB
  • You control the class names of the scrollbar elements to allow any combination of scrollbar styles you need on the same page
  • Preserve semantics — you don’t need to add any extra tags in your markup. Since enscroll adds these client-side, you can keep the initial DOM flatter
  • There are other advanced features you can discover for yourself in the documentation …

Usage

The usual way to set up the enscroll plugin:

textpop-up
  1. $(”#my_scrolling_pane”).enscroll({
  2.  
  3.     // a configuration property
  4.     showOnHover: true,
  5.  
  6.     // another configuration property
  7.     verticalScrolling: true
  8.  
  9.     // ... more configuration properties ...
  10. });
$(”#my_scrolling_pane”).enscroll({

	// a configuration property
	showOnHover: true,

	// another configuration property
	verticalScrolling: true

	// ... more configuration properties ...
});

Configuration

A complete list of available configuration properties:

Property Default Value Description
verticalScrolling true Whether to render the scrollbars on the right to scroll vertically when the content of the view pane has a height greater than the height of the view pane itself
horizontalScrolling false Whether to render the scrollbars on the bottom to scroll horizontally when the content of the view pane has a width greater than the width of the view pane itself
showOnHover false When enabled, the scrollbar(s) will remain hidden until the user hovers the mouse cursor over the view pane. This feature is useful when creating a scrolling UI similar to Facebook’s ticker on the News Feed
scrollIncrement 20 The distance(in pixels) the view pane should scroll when using the mousewheel or arrow keys on the keyboard
minScrollbarLength 25 The length of the scrollbar handles will vary in length according to the size of the content in the view pane. This property specifies a minimum length(in pixels) beyond which the scrollbar will not contract
pollChanges true Whether to listen for changes to the view pane including content being changed within the view pane, changes to the width and height of the view pane, and changes to the position of the view pane. If one of these changes is detected, the scrollbar(s) will be updated to reflect these new properties
verticalTrackClass> 'vertical-track' The CSS class name given to the track of the vertical scrollbar
horizontalTrackClass 'horizontal-track' The CSS class name given to the track of the horizontal scrollbar
verticalHandleClass 'vertical-handle' The CSS class name given to the handle of the vertical scrollbar
horizontalHandleClass 'horizontal-handle' The CSS class name given to the handle of the horizontal scrollbar
